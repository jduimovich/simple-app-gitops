Simple-App GitHub Actions Demo

Leverage OpenShift Developer Tools and GitHub Actions

Red Hat GitHub Actions Workflow is designed to help you work seamlessly with Red Hat Developer tools and Openshift clusters.

This application used the RedHat OpenShift Starter Workflow and OpenShift Gitops to get an app running in minutes.

Together, we're super awesome

Run an App using OpenShift Gitops in 4 Easy Steps

  1. Modify the `argo` directory to point to your fork.
  2. `oc apply -f argo` to install the application on your cluster
  3. The app will be running on your cluster.
  4. Push changes to your app and the built-in workflow will create a pull-request or will directly push changes into your gitops configuration. See the workflow to set the default or include 
    /update-via-push
    or 
    /update-via-pr
    in your commit message to override the default in your workflow.
